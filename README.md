# Анализ задержек авиарейсов: причины, авиакомпании и маршруты

**Данные:** BTS On‑Time Performance (Министерство транспорта США), 2024 год  
**Объём:** 7+ млн записей, 35 колонок  
**Инструменты:** Python (pandas, matplotlib, numpy), Jupyter Notebook

---

## Цель проекта

Выявить основные причины задержек авиарейсов, определить самых непунктуальных перевозчиков и маршруты с наибольшей долей задержек.

---

## Ключевые результаты

### 1. Общая статистика задержек

Статистика задержки вылета:
Среднее время: 13.0 минут.
Медиана: -2.0 минут. (половина рейсов вылетают раньше расписания)
Стандартное отклонение: 53.63 минут.
Максимальное время задержки: 2011.0 минут.

Статистика задержки прилета:
Среднее время: 7.51 минут.
Медиана задержки: -6.0 минут. (половина рейсов прилетают раньше расписания)
Стандартное отклонение: 55.68 минут.
Максимальное время задержки: 2014.0 минут.

**Вывод:** большинство рейсов пунктуальны, но есть небольшое число очень долгих задержек, которые создают длинный правый хвост распределения.

### 2. Авиакомпании с наибольшей задержкой (> 1000 рейсов)
| Авиакомпания | Код | Средняя задержка прилёта | Медиана |
|--------------|-----|--------------------------|---------|

---

### 3. Маршруты с высокой долей задержек
**При пороге > 18 рейсов (системные проблемы):**
- LAX–LAS: 21.1% задержек > 15 мин
- Остальные маршруты: < 10%

**При пороге > 10 рейсов (локальные аномалии):**
- ATL–DFW: 63.6% задержек > 15 мин
- SAN–SFO: 45.5%
- LAX–DFW: 41.7%

**Вывод:** системных проблем с задержками на маршрутах нет, но есть локальные всплески, требующие дополнительного анализа.

### 4. Причины задержек
**Вклад в общее время задержек:**
| Причина | Доля |
|---------|------|
| late_aircraft_delay | 40% |
| carrier_delay | 32% |
| nas_delay | 20% |
| weather_delay | 7% |
| security_delay | < 1% |

**Средняя длительность задержки (только задержанные рейсы):**
| Причина | Средняя задержка |
|---------|------------------|
| weather_delay | 76.8 мин |
| late_aircraft_delay | 57.7 мин |
| carrier_delay | 42.9 мин |
| nas_delay | 28.7 мин |
| security_delay | 14.7 мин |

**Вывод:** 72% задержек связаны с двумя причинами — задержка предыдущего рейса и проблемы авиакомпании. Основные рычаги влияния на пунктуальность — внутри авиакомпаний и их операционной логистики.
